## About 
A ReactNative app with TypeScript template. This app fetches and displays a list of events from github.io api.

## Install

```bash
yarn
//or
npm install
```


## Starting emulator

start server

```bash
yarn start
//or
npx react-native start
```

and run emulator

```bash
yarn android
//npx react-native run-android

//or

yarn ios
//npx react-native run-ios
```
